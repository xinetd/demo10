# demo10
A simple Spring Boot REST based CRUD server

# design
+ Spring Boot (@RestService)
+ Spring Data JPA 
+ H2 in memory DB
+ Spring MVC (Middle ground between unit/integration test)

# build
mvn package

# execute
mvn spring-boot:run or
java -jar target/demo10-0.0.1-SNAPSHOT.jar

# test
mvn test
